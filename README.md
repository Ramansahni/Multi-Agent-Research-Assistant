# Multi-Agent Research Assistant

A multi-agent AI research assistant built using LangChain, Mistral AI, Tavily Search, BeautifulSoup, and Streamlit.

## Features

- Multi-agent research workflow
- Web search using Tavily
- Intelligent webpage scraping
- AI-generated research reports
- AI critic for report evaluation
- Interactive Streamlit interface

## Tech Stack

- Python
- LangChain
- Mistral AI
- Tavily Search API
- BeautifulSoup
- Streamlit

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
